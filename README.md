# Plataforma Web Inteligente para Recursos Audiovisuales

Proyecto de título desarrollado como parte de la Evaluación Sumativa N°3.

## Descripción
Sistema web orientado a la gestión y optimización de recursos audiovisuales mediante el registro de préstamos, control de disponibilidad y análisis de uso.

## Funcionalidades
- Inicio de sesión de usuarios
- Gestión de recursos audiovisuales
- Solicitud de préstamos
- Registro de devoluciones
- Visualización de recursos disponibles
- Estadísticas de uso

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- Node.js
- MySQL

## Base de datos
El archivo `basedatos.sql` contiene la estructura de la base de datos utilizada en el proyecto.
